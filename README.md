# Epistemic orientation in parliamentary discourse is associated with deliberative democracy

This repository contains the code used for the analysis and results reported in the manuscript **Epistemic orientation in parliamentary discourse is associated with deliberative democracy**


## Notes
* We developed the codes using Python (3.11.6) and R (4.3.1) on Ubuntu 22.04.
* The aggregated EMI scores and data required to reproduce the plots and statistical analyses are available in the `aggregated_data` directory, with corresponding scripts in the `analysis` directory.
* The scripts in the labeling directory require input data that are hosted separately on Zenodo due to their size [data repository](https://doi.org/10.5281/zenodo.19666593).
    *  Python package dependencies can be installed using the `requirements.txt` file i.e., `pip install -r requirements.txt`.
    *  Hardware note: We ran the labeling scripts on a machine equipped with an H100 GPU.
